# 15. Glossary of Terms

| Term | Definition in this document's context |
|---|---|
| RBAC | Role-Based Access Control, granting permissions based on a scoped role rather than to an individual directly. |
| MFA | Multi-Factor Authentication, requiring more than one proof of identity to log in. |
| CVE | Common Vulnerabilities and Exposures, a public identifier for a known security flaw. |
| CADF | Cloud Auditing Data Federation, the audit-event format Keystone, Nova, and Neutron use for logging actions. |
| CephX | Ceph's built-in authentication protocol between its own daemons and clients. |
| VLAN | Virtual Local Area Network, a way of logically separating network traffic on shared physical hardware. |
| TLS | Transport Layer Security, the protocol that encrypts data while it is in transit across a network. |
| SIEM | Security Information and Event Management, a system that ingests and correlates security-relevant events from many sources. |
| ISMS | Information Security Management System, the formal program ISO/IEC 27001 certifies an organization for operating. |
| OSSN | OpenStack Security Note, an official OpenStack security advisory. |
| Zero Trust | A security approach where no request is automatically trusted based on its network location alone. |
| Defense in Depth | Layering multiple independent security controls so no single failure results in full compromise. |
